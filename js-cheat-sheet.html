Сравнения

x == y

Возвращает True если x и y одинаковы

x === y

Возвращает True если x = y

x != y

Возвращает True если x и y неодинаковы

x !== y

Возвращает True если x ≠ y

x > y

Возвращает True если x больше чем y

x >= y

Возвращает True если x больше или равно y

x < y

Возвращает True если x меньше чем y

x <= y

Возвращает True если x меньше или равно y

x && y

Возвращает True если x и y правдивы

x || y

Возвращает True если или x или y правдиво

!x

Возвращает True если x неверно
